## 📖 About the Project

This landing page is designed for **Chai Culture**, reflecting a premium and warm brand identity inspired by Indian chai traditions. The goal is to drive engagement and build a subscriber base before the official launch.

![Demo Preview](./demo.png)

### ✨ Features
- **Clean Split Layout:** A modern, aesthetically pleasing dual-pane design.
- **Earthy Color Palette:** Warm colors that evoke the feeling of traditional Indian chai.
- **Dynamic Countdown:** A high-quality JavaScript countdown timer tracking to the launch date.
- **Engaging Form:** Built-in email subscription form to capture early interest.
- **Fully Responsive:** Seamless viewing experience across all mobile and desktop devices.

## 🚀 Usage

To view this project locally, you don't need any complex build steps. Simply:

1. Clone or download the repository.
2. Open `index.html` in your favorite web browser.
